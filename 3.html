JavaScript

JavaScript(English: JavaScript)is an object-based script programming language. This language is commonly used within a Web browser and has the ability to access built-in objects from other applications. It is also used for server programming, such as runtime environments such as Node.js.JavaScript was originally developed by BrendanEich of Netscape Communications Corporation, first named Mocha and later liveScript, and eventually became JavaScript.
